# iNotePal

**AI와 대화를 통한 매일 감정 기록 노트**

iNotePal은 감성 대화 챗봇과의 대화를 통해 사용자의 하루를 기록하고, 그 날의 감정을 분석하여 시각화해주는 웹 애플리케이션입니다. 기존의 딱딱한 자기보고식 검사 대신, 자연스러운 대화를 통해 사용자의 정서 상태를 파악하고 기록합니다. <a href="https://github.com/pal-ette/iNotePal" target="_blank" class="no-underline"><Icon name="github" class="inline-icon" /> 소스 코드</a>

## 주요 기능

-   **감성 대화 챗봇**: 사용자의 말에 공감하고 반응하는 챗봇과 대화할 수 있습니다.
-   **감정 분석**: 대화 내용을 분석하여 기쁨, 슬픔, 분노 등 7가지 주요 감정으로 분류합니다.
-   **감정 시각화**: 기간별 감정 변화를 그래프와 차트로 시각화하여 보여줍니다.
-   **캘린더 뷰**: 달력 형태로 매일의 감정 기록을 한눈에 확인할 수 있습니다.

## 프로젝트 목표

-   자연스러운 대화를 통해 사용자의 정서 상태를 정확하게 파악.
-   매일의 감정을 기록하고 시각화하여 사용자가 자신의 마음을 돌볼 수 있도록 지원.
-   의료 상담 등에서 활용 가능한 진단 가치 창출.
