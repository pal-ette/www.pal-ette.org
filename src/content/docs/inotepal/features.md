# 앱 기능

## 1. 감성 대화 (Chat)
-   **자연스러운 대화**: AI 챗봇과 일상적인 대화를 나눌 수 있습니다.
-   **공감 반응**: 사용자의 감정에 맞춰 위로하거나 공감하는 답변을 제공합니다.

## 2. 감정 대시보드 (Dashboard)
-   **감정 분포**: 오늘, 이번 주, 이번 달의 감정 분포를 파이 차트 등으로 확인합니다.
-   **워드 클라우드**: 대화에서 자주 사용된 단어들을 시각적으로 보여주어, 사용자의 주요 관심사나 고민을 파악할 수 있게 합니다.
-   **감정 흐름**: 시간의 흐름에 따른 감정 변화 추이를 꺾은선 그래프로 제공합니다.

## 3. 캘린더 (Calendar)
-   **감정 일기**: 달력의 날짜별로 그날의 주된 감정을 이모티콘이나 색상으로 표시합니다.
-   **기록 조회**: 특정 날짜를 클릭하면 그날의 대화 내용과 감정 분석 결과를 다시 볼 수 있습니다.

## 4. 사용자 맞춤 분석
-   **기간별 분석**: 사용자가 지정한 기간 동안의 정서 상태를 분석하여 제공합니다.
-   **진단 가치**: 이러한 기록들은 추후 심리 상담이나 의료적 진단에 참고 자료로 활용될 수 있습니다.
